# AI-Assisted React Homework Assignment

This project is a small React task manager built as a homework submission example. It demonstrates how AI can help scaffold a UI, suggest structure, and speed up implementation while still requiring manual refinement.

## What the app includes
- Add tasks
- Mark tasks complete or active
- Filter tasks by all, active, or completed
- Delete tasks
- Clear completed tasks
- Persist tasks in local storage

## Prompts used
1. "Create a simple React task manager app with add, complete, delete, and filter features."
2. "Make the UI look polished and modern for a homework submission."
3. "Add local storage so tasks remain after a page refresh."
4. "Include a short section explaining the AI-assisted workflow for the assignment."

## How AI helped
AI helped by generating the initial component structure, the interactive state logic, and a first-pass polished UI. The manual improvements included refining accessibility, adjusting spacing, and ensuring the final experience felt clearer and more complete.

## Manual improvements made
- Added accessible labels and ARIA attributes.
- Improved layout and spacing for readability.
- Verified the app builds and runs correctly.

## Run locally
```bash
npm install
npm run dev
```
